# Benchmark corpora

These files are excerpts of public-domain literary works, used by `benches/throughput.rs` as realistic running-text input for measuring `check()` throughput. They are included purely as benchmark data.

## Sources and attribution

Each file is an excerpt (about 50 KB) of a Project Gutenberg plain-text edition. The Project Gutenberg license boilerplate (header and footer) has been removed, leaving only the underlying work, which is in the public domain.

- `english.txt`: Jane Austen, _Pride and Prejudice_ (1813). Source: Project Gutenberg eBook #1342.
- `french.txt`: Voltaire, _Candide, ou l'optimisme_ (1759). Source: Project Gutenberg eBook #4650.
- `burns.txt`: Robert Burns, _Poems and Songs_ (Scots dialect). Source: Project Gutenberg eBook #1279.

All three works are in the public domain in the United States, so no license accompanies them. The credits above are provided for provenance.
